# Mailfold frontend

React single-page app for Mailfold.

> **Status:** awaiting the design project. The UI will be scaffolded from the
> incoming design; this directory is a placeholder until then.

## Planned stack

- React + TypeScript
- Vite build tooling
- Talks to the Go backend under `/api/*`

## Conventions

- All code and comments in English.
- The production build outputs to `frontend/dist/`, which the Go backend serves
  (see `MAILFOLD_FRONTEND_DIR`).
